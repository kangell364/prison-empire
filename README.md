# Prison Empire 🏛️

A mobile-first strategy game where you run your crew from inside prison.

## Game Features
- **Prison Hierarchy** — County Jail → State Prison → Federal Penn → Supermax
- **Dual Gameplay** — Manage operations inside AND outside simultaneously  
- **City Map** — Control real US cities with your outside crew
- **Attack Timer** — 15-minute attack windows with snitch protection
- **Battle Cards** — Collect and level up character cards
- **Boss Battles** — Fight through areas to unlock new content
- **Leaderboard** — Monthly transfers based on ranking

## Tech Stack
- React 18
- CSS Variables for theming
- Mobile-first design (390px)

## Getting Started

```bash
npm install
npm start
```

## Screens
- **Home** — Dashboard with resources, crew, city overview
- **Map** — Texas territory map, attack cities
- **Fight** — Card battles vs AI enemies and bosses
- **Cards** — Collection, pack opening, card details
- **Profile** — Player stats and settings
